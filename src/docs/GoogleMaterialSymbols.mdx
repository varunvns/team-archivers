This project includes Google Material Symbols. For additional information on the Google Material Symbols library, visit <a href="https://fonts.google.com/icons">fonts.google.com/icons</a>.

Icons can be included in any React component by including the following code snippet:

<GoogleMaterialSymbol icon="cached" />
`}
